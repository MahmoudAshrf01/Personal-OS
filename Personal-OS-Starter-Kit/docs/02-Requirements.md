# Requirements

## Functional

- **Tasks** — CRUD, statuses (planned / in progress / done), priorities, due dates, tags, DnD board
- **Nested Groups** — hierarchical task/goal grouping
- **Goals** — long-term objectives linked to tasks
- **Calendar** — due-date overview
- **Notes** — rich-text notes (TipTap)
- **Journal** — dated journal entries
- **Pomodoro** — focus timer with session tracking
- **Time Tracking** — log focus/work sessions
- **Gamification** — XP, levels, coins, rewards, achievements, streaks
- **Analytics** — completion rates, XP growth, heatmap, focus time
- **Reviews** — weekly/monthly reflection
- **Snapshot Engine** — export/import state snapshots
- **Search & Archive** — global search, archived items

## Non-Functional

- React + TypeScript + Vite
- Tailwind CSS + Kibo UI + shadcn/ui
- **Motion** for animations (`motion/react`)
- Zustand for UI state
- Dexie (IndexedDB) for persistence
- Offline first
- Backend-ready architecture (repository pattern)
- AMOLED-friendly dark theme
- Accessible (WCAG-oriented) & keyboard navigable
