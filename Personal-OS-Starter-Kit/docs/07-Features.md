# Features

| Module | Route | Description |
|--------|-------|-------------|
| **Dashboard** | `/` | Overview — stats, XP, quick links |
| **Tasks** | `/tasks` | Kanban board with DnD, filters, search |
| **Goals** | `/goals` | Long-term objectives with progress |
| **Calendar** | `/calendar` | Due-date task overview |
| **Notes** | `/notes` | Rich-text notes (TipTap-ready) |
| **Journal** | `/journal` | Dated journal entries with mood |
| **Pomodoro** | `/pomodoro` | 25-minute focus timer |
| **Analytics** | `/analytics` | Completion rates, heatmap bars |
| **Gamification** | `/gamification` | XP, levels, coins, achievements |

## Cross-cutting

- **Search & Archive** — task search in toolbar; archive via status
- **Nested Groups** — group repository supports `parentId`
- **Snapshots** — snapshot engine captures full state
- **Motion** — page transitions and list animations on all modules
