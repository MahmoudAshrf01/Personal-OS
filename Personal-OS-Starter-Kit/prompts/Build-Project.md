# Build Project Prompt

Follow **all docs** in `Personal-OS-Starter-Kit/docs/` (01–19) and build the project module by module.

## Order

1. Read `03-Tech-Stack.md` — install deps including **Motion** (`motion/react`)
2. Scaffold folder structure per `11-Folder-Structure.md`
3. Run `Build-Database` prompt — Dexie + repositories
4. Run `Build-Tasks` prompt — task board with DnD
5. Run `Build-Gamification` prompt — XP, levels, coins
6. Run `Build-Analytics` prompt — dashboards
7. Add remaining feature modules (Goals, Notes, Journal, Calendar, Pomodoro)
8. Wire React Router app shell with sidebar navigation
9. Apply AMOLED dark theme and Motion animations per `14-UI-UX-Rules.md`
10. Verify `npm run build` passes

## Do not skip

- Repository pattern (backend-ready)
- Engine layer for business logic
- Motion on page transitions and list items
- Offline-first Dexie persistence
- Gamification hooks on task lifecycle events
